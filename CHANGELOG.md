# Changelog

Все заметные изменения проекта документируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект придерживается [семантического версионирования](https://semver.org/lang/ru/).

## [1.1.0] — 2026-06-18

### Добавлено
- **Конфиги ИИ-агентов и инструментов** в сканировании AppData: Claude Desktop/Code (`.claude`), Cursor, Windsurf, Codeium, Continue, Zed, Ollama, LM Studio, Jan, GPT4All, Msty, ChatGPT, AnythingLLM, Aider — попадают в «⭐ Рекомендуемые».
- Категория **«🤖 ИИ-инструменты»** в списке установленных программ (с переводом на EN).
- Раздел **«🤖 ИИ-агенты и инструменты»** на экране рекомендаций.
- **Поиск** на экране рекомендаций — фильтр по названию и описанию.
- **Прогресс сканирования в процентах** вместо бесконечной полосы.
- **Самодокументируемая структура резервной копии** — зеркалит папку пользователя (`configs\AppData\Roaming\Code` → `%USERPROFILE%\AppData\Roaming\Code`).
- Файл **`КАК_ВОССТАНОВИТЬ.txt`** (RU/EN) в каждой копии — инструкция и карта «что → куда» для ручного восстановления.

### Изменено
- **Архивирование переписано**: пути в ZIP теперь только с прямыми слэшами (раньше на Windows попадали обратные — архив открывался «криво»); восстановление из архива распаковывает файлы напрямую в папку пользователя без промежуточного раздувания; плавный прогресс и поддержка отмены на больших папках.
- Единая цветовая палитра — экран рекомендаций, статусбар и окно логов приведены к основной теме.
- Убрана неиспользуемая зависимость `py7zr` (архивы — ZIP через стандартный `zipfile`).

### Исправлено
- 🔴 Личные **папки** в режиме «копировать» не восстанавливались (рассинхрон путей copy/restore).
- 🔴 **SSH-ключи** и одиночные файлы-конфиги восстанавливались во вложенную папку (`~/.ssh/id_rsa/id_rsa`) вместо файла.
- 🔴 Два файла с одинаковым именем из разных папок перезаписывали друг друга в копии.
- Ложные метки рекомендаций по подстроке (папка «Digital…» помечалась как Git, короткие `amd`/`vlc` ловили случайные совпадения).
- Шаг 1 переанализировал диски при возврате «Назад», затирая введённые папку и название сессии.
- Вкладка «Программы» не локализовалась в английском режиме.

### Тесты
- Добавлены roundtrip-тесты `copy`/`archive` → `restore` ([test_file_operations.py](test_file_operations.py)): зеркальная структура, SSH-ключи, конфиг-папки, коллизии имён, генерация инструкции.
